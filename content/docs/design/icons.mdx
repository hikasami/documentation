## Использование

### Динамическая загрузка
```typescript
import { Icon } from '@/shared/icons/Icon';

<Icon name="heartregular" width={24} height={24} />
```

### Прямой импорт (без ленивой загрузки)
```typescript
import { HeartRegularIcon } from '@/shared/icons';

<HeartRegularIcon width={24} height={24} color="red" />
```

### Стилизация

```scss
// Иконки наследуют цвет от родителя
.button {
  color: var(--color-primary);
  
  .icon {
    // Иконка будет цвета --color-primary
  }
}
```
