## Фоновые цвета
Для базовых страниц и контейнеров предусмотрено два фоновых цвета. **Primary** используется для главного фона (`html`, `body`, основные блоки), **Secondary** — для вложенных элементов, карточек и модальных окон.

## Границы

Для отделения компонентов в интерфейсе используются полупрозрачные границы.  
Основным является `rgba(250, 250, 250, 0.04)`. Он применяется, когда нужно **ярко и чётко выделить объект**, при этом сохраняя лёгкость интерфейса.

<HikasamiColorList
  colors={[
    {
      name: "Граница 1",
      variable: "rgba(250, 250, 250, 0.04)",
      description: "Основной бордер для чёткого выделения элементов",
    },
    {
      name: "Граница 2",
      variable: "--ds-border-primary",
      description: "Альтернативный бордер для менее заметных разделителей",
    },
  ]}
/>

### Зачем используется полупрозрачная граница?

- **Визуальная лёгкость** — бордер не перегружает интерфейс и не создаёт жёстких границ.  
- **Адаптивность** — прозрачность позволяет бордеру подстраиваться под фон.  
- **Единый стиль** — поддерживает философию «тепло среди холода»: видимый, но не навязчивый.  
- **Чистота интерфейса** — в списках и таблицах не возникает «сетчатого эффекта».  
- **Баланс в тёмной теме** — цвет не слишком контрастный и не теряется на фоне.  

Таким образом, `rgba(250, 250, 250, 0.04)` работает не как отдельный цвет, а как **слой подсветки**, который делает границы более органичными.

## Фоны компонентов

Для пользовательских интерфейсных компонентов предусмотрено три фоновых состояния.  
Они помогают визуально разделять стандартное, наведение и активное состояние элементов.

<HikasamiColorList
  colors={[

    {
      name: "Фон компонента 2",

  ]}

## Текст и иконки

В дизайне используется ограниченная палитра для текста и иконок, чтобы сохранить читаемость и единый стиль.  
Основная идея — минимализм и акцент на важном.

## Серый
Серый используется как вспомогательная палитра для текста, иконок и системных элементов.  
В ней учтены связи с основными фонами:  
- `gray-900` совпадает с **Фоном 1**  
- `gray-700` совпадает с **Фоном 2**  
- `gray-600` совпадает с **Границей 2**
